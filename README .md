# 🔍 4PDA Attachments Parser

Юзерскрипт для браузера, который автоматически находит и собирает все вложения (книги, архивы, APK-приложения) на страницах форума [4pda.to](https://4pda.to).

## ✨ Возможности

- **Сканирование страницы** — находит все ссылки на вложения на текущей странице форума
- **Гибкая фильтрация** по типам файлов:
  - 📄 PDF-документы
  - 📚 Электронные книги: EPUB, FB2, DJVU, MOBI, AZW
  - 📦 Архивы: ZIP, RAR, 7Z, TAR, GZ
  - 📱 Android-приложения: APK, APKS, XAPK, APKM
- **Экспорт в CSV** — сохранение найденных файлов с именами, типами и ссылками
- **Копирование ссылок** — все найденные URL одним кликом в буфер обмена
- **Счётчики** — показывает количество найденных файлов и проверенных ссылок

## 📦 Установка

1. Установите расширение для управления юзерскриптами:
   - [Tampermonkey](https://www.tampermonkey.net/) (Chrome, Firefox, Edge, Safari)
   - [Violentmonkey](https://violentmonkey.github.io/) (Chrome, Firefox)

2. Нажмите на кнопку ниже, чтобы установить скрипт:

   > [📥 Установить скрипт](../../raw/main/4pda-parser_user.js)

   *(или откройте файл `4pda-parser_user.js` вручную в менеджере скриптов)*

## 🚀 Использование

1. Перейдите на любую страницу форума **4pda.to** или **4pda.ru**
2. В правом верхнем углу появится кнопка **🔍**
3. Нажмите на неё, чтобы открыть панель парсера
4. Выберите нужные типы файлов в настройках фильтров
5. Нажмите **«Сканировать страницу»**
6. После сканирования:
   - Нажмите **«Экспорт в CSV»** — чтобы скачать список найденных файлов
   - Нажмите **«Копировать ссылки»** — чтобы скопировать все URL в буфер обмена

## 🖥️ Интерфейс

| Элемент | Описание |
|---|---|
| 🔍 Кнопка (правый верхний угол) | Открывает/закрывает панель парсера |
| Счётчик «Файлов найдено» | Количество файлов, соответствующих выбранным фильтрам |
| Счётчик «Ссылок проверено» | Общее число проверенных ссылок на странице |
| Фильтры | Выбор типов файлов для поиска |
| Кнопка «Сканировать» | Запуск сканирования текущей страницы |
| Кнопка «Экспорт в CSV» | Сохранение результатов в CSV-файл |
| Кнопка «Копировать ссылки» | Копирование всех URL в буфер обмена |

## 📋 Формат CSV-экспорта

Экспортируемый файл содержит следующие колонки:

| Файл | Тип | Тема | URL файла | URL темы |
|---|---|---|---|---|
| book.epub | EPUB | Название темы | https://... | https://... |

Файл сохраняется с именем вида `4pda_attachments_<timestamp>.csv` в кодировке UTF-8.

## 🔧 Технические детали

- **Версия:** 1.2.0
- **Совместимость:** 4pda.to, 4pda.ru (все разделы форума)
- **Разрешения:** `GM_setClipboard`, `GM_download`
- Скрипт использует расширенный набор CSS-селекторов для поиска вложений, включая блоки `.post-attach`, `.attachunit`, `.postattach`, `.attachment` и другие
- Дублирующиеся ссылки автоматически исключаются

## 📝 Поддерживаемые форматы файлов

| Категория | Форматы |
|---|---|
| Документы | `.pdf` |
| Книги | `.epub`, `.fb2`, `.djvu`, `.mobi`, `.azw`, `.azw3` |
| Архивы | `.zip`, `.rar`, `.7z`, `.tar`, `.gz` |
| Android | `.apk`, `.apks`, `.xapk`, `.apkm` |
